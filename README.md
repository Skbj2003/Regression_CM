# Sri Lanka River Water Level Forecasting 🌊

This repository contains my statistical and machine learning models for predicting river water levels in Sri Lanka. It was created as part of a statistics/data science assignment.

## Project Overview

The goal of this project is to build regression models (including Univariate and Multiple Linear Regression) to accurately forecast water levels using historical training data. 

## Repository Contents

*   **Jupyter Notebooks**:
    *   `UnivariateModel.ipynb`: Initial exploration and univariate regression modeling.
    *   `submission_improved.ipynb` / `submission_final.ipynb`: Iterative improvements and the final Multiple Linear Regression models.
*   **Data**: 
    *   `training_data.csv` and `test_data.csv` for training and evaluating the models.
*   **Documentation**:
    *   `Assignment1.pdf`: Assignment guidelines and requirements.
*   **Results**:
    *   Various `submission.csv` files generated for evaluation.

## Tools Used
*   Python
*   Pandas & NumPy (Data Manipulation)
*   Scikit-Learn / Statsmodels (Regression Modeling)
*   Jupyter Notebook (Environment)
